# 🎁 Gift Expert Agent - Project Context

## 🎯 Project Goal
Create a Fetch.ai agent called "gift-expert" that responds to any message starting with "@" with the phrase "Hey, I'm a gift expert" for ASI:One integration.

## 🏗️ Architecture Overview
- **Local Development**: Agent runs on localhost:8001
- **Public Access**: ngrok tunnel provides public URL
- **Agentverse Integration**: Agent registered with Fetch.ai's Agentverse platform
- **ASI:One Integration**: Agent responds to @mentions in ASI:One chat

## 🔧 Technical Stack
- **Framework**: Fetch.ai uAgents
- **HTTP Server**: FastAPI + Uvicorn (for ASI:One compatibility)
- **Tunnel**: ngrok (for public access)
- **Platform**: Agentverse (agent marketplace)
- **Integration**: ASI:One (chat platform)

## 📁 Key Files
